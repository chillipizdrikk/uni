Реалізація задач зі стеком

Опис:

В даній лабораторній реалізовано клас ProtectedStack та розв'язані задачі по роботі зі стеком відповідно до вимог:

- Всі задачі нерекурсивні (де це вимагається).
- Власна реалізація стеку (без collections, без сторонніх модулів).
- Стек захищений: доступ лише через методи класу.
- Для задач 1.1–1.3 не використовується жодне сортування (ні явне, ні неявне).
- Для кожної задачі — окремий файл з тестами.
- Тести запускаються через зміну `test_case_index` у відповідному файлі.

Список файлів:

- `protected_stack.py` — клас ProtectedStack.
- `task_1_1.py` — Задача 1.1: пошук пар круглих дужок.
- `task_1_2.py` — Задача 1.2: пари дужок за зростанням позицій відкриваючих дужок.
- `task_1_3.py` — Задача 1.3: пари дужок за зростанням позицій закриваючих дужок.
- `task_2.py` — Задача 2: сума елементів у вкладених списках (без рекурсії).
- `task_3.py` — Задача 3: перевірка правильності формули з дужками ((), [], {}).
- `task_4.py` — Задача 4: обчислення значення формули S(x,y), D(x,y).
- `task_5_1.py` — Задача 5.1: Ханойська вежа (ітеративно, через стеки).
- `task_5_2.py` — Задача 5.2: сортування елементів на стеку.

Запуск:

1. Для запуску задачі — відкрийте відповідний файл та запустіть його.
2. Для вибору іншого тесту змініть значення `test_case_index` в потрібному файлі.
3. Всі задачі імпортують стек з `protected_stack.py`.

Тести:

- У кожному файлі реалізовано набір тестів для граничних та типових випадків.

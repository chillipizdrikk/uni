1) Вхідні дані:
   - Один XML-файл, отриманий з інтернету (наприклад, щоденний файл курсів валют від ECB або RSS/ATOM-стрічка).

2) Вимоги:
   2.1 Детектувати формат:
      - Якщо в файлі є елементи <Cube time="YYYY-MM-DD"> — вважати, що це файл курсової таблиці (ECB).
      - Якщо є елементи <item> або <entry> то вважати, що це RSS/ATOM-стрічка.
   2.2 Для файлу типу "курси валют" (ECB):
      - Витягти дату курсу з атрибуту time.
      - Побудувати CSV-файл з колонками: currency, rate, date.
      - Підрахувати кількість валют (включаючи EUR як базову валюту зі ставкою 1.0).
      - Зберегти модифікований XML з атрибутом processed_by і processed_at у кореневий елемент.
   2.3 Для RSS/ATOM:
      - Вивести список перших N записів (N = 10), кожен рядок: заголовок, посилання, дата публікації.
      - Підрахувати кількість записів.
      - Зберегти модифікований XML з додатковими атрибутами processed_by і processed_at.
   2.4 Загальні завдання:
      - Реалізувати трасування: друкувати кроки виконання, кількість знайдених вузлів, повідомлення про помилки.
      - Реалізувати запис у новий файл (CSV або XML) у тій самій директорії.

3) Формат результатів:
   - CSV та/або XML-файли у тій самій папці.
   - Лог виконання - вивід у стандартний потік (stdout), який буде включати інформацію про кожний крок.

5) Приклади запуску:
   - python task4_processing.py ecb_rates.xml
   - python task4_processing.py feed.xml

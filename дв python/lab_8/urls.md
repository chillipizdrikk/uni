# Веб-адреси використані у прикладах

1) https://www.python.org/
   - Тип: веб-сторінка (HTML)
   - Опис: Офіційний сайт Python — головна сторінка проекту. Містить статичну й динамічну інформацію, новини, посилання на завантаження. Використовується як приклад завантаження веб-сторінки.

2) https://www.ecb.europa.eu/stats/eurofxref/eurofxref-daily.xml
   - Тип: XML (дані)
   - Опис: Щоденний файл з курсами валют Європейського центрального банку (ECB). Містить базовий курс EUR та курси інших валют на певну дату.

3) https://earthquake.usgs.gov/earthquakes/feed/v1.0/summary/all_day.atom
   - Тип: Atom (XML) — стрічка подій (feed)
   - Опис: Підсумок землетрусів за останні 24 години від USGS у форматі Atom (XML). Містить записи (entry) з координатами, магнітудою, часом.

4) https://feeds.bbci.co.uk/news/rss.xml
   - Тип: RSS (XML)
   - Опис: RSS-стрічка новин від BBC. Містить елементи <item> з заголовками, описами, посиланнями та датами.

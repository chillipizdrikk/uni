Короткий опис проєкту:

Проєкт моделює роботу пункту прикордонного митного контролю для вантажних автомобілів з використанням власного класу черги. Реалізовано базові операції черги, операції, пов'язані з митним процесом, та засоби контролю і захисту даних. Використано структуру dataclass для шаблону митної декларації та перелік умовних декларацій (18 записів).

Файли:
- queue.py - реалізація класу Queue з базовими методами, методами для митного процесу та механізмами валідації/захисту.
- declarations.py - dataclass Declaration та список incoming_declarations (18 декларацій).
- customs_simulation.py - основний сценарій (диспетчер), демонструє використання всіх методів, записує протокол у protocol.txt.
- protocol.txt - зразок виводу протоколу, створеного під час запуску сценарію.

Запуск:

1. Помістіть файли queue.py, declarations.py, customs_simulation.py в одну теку.
2. Для запуску відкрийте файл customs_simulation.py та запустіть його.
3. Після виконання протокол буде збережено у protocol.txt. Консоль одночасно показує повідомлення та вони дублюються в protocol.txt.

FileToolsTkinter - Інструменти для роботи з файлами і каталогами (tkinter)

Опис

FileToolsTkinter - це невеликий графічний застосунок на Python (tkinter), який виконує чотири завдання по роботі з файлами і каталогами:
- Завдання 1: знайти два найдавніші і два найновіші файли в папці (імена + дати).
- Завдання 2: знайти дублікати (за ім'ям і розміром) у двох папках.
- Завдання 3: перерахувати файли даного типу (.ext) у вкладених підпапках першого рівня.
- Завдання 4: розподілити (скопіювати або перемістити) файли у папці по підпапках за розширенням.

Файли в репозиторії
- `FileToolsTkinter.pyw` - основний скрипт застосунку (GUI).
- `README.md` - цей файл з інструкцією.

Як використовувати?

1. Для запуску відкрийте файл FileToolsTkinter.pyw та запустіть його.
 
2. Створіть sandbox:
   - У меню виберіть "Sandbox -> Create sandbox and sample data".
   - За замовчуванням створюється папка `~/file_tools_sandbox`.

3. Згенеруйте тестові дані:
   - Натисніть на кнопки "Create sample for Task1/Task2/Task3/Task4" - вони створять демонстраційні папки і файли в `~/file_tools_sandbox`.
   - Після генерації можна натиснути "Open sandbox folder" -> щоб переглянути файли у файловому менеджері.

4. Запустіть потрібне завдання:
   - Task 1: натисніть кнопку "Task 1: Oldest/Newest" -> виберіть папку (наприклад `~/file_tools_sandbox/task1`) -> результати з'являться у текстовій області.
   - Task 2: натисніть "Task 2: Find duplicates" -> виберіть дві папки (folderA і folderB) -> програма виведе знайдені дублікати (ім'я + розмір + шляхи).
   - Task 3: натисніть "Task 3: List by type" -> виберіть батьківську папку (наприклад `~/file_tools_sandbox/task3`) -> введіть розширення (наприклад py або txt) -> отримаєте список відповідних файлів у підпапках.
   - Task 4: натисніть "Task 4: Split by type" -> виберіть папку (наприклад `~/file_tools_sandbox/task4`) -> за замовчуванням увімкнено "Simulate" (копіювання). Якщо вимкнути - програма буде переміщувати файли (руйнівна дія); перед цим відобразиться підтвердження.

5. Експорт результатів:
   - Натисніть "Export results to results.txt" або в меню "Export results" -> вкажіть шлях для збереження файлу.

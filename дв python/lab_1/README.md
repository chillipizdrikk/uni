# Тестування функцій (Задачі 1-4)

# Файли
`FnSolutions.py` — модуль з усіма функціями та організацією тестування
`InData1.txt`, `InData2.txt`, `InData3.txt`, `InData4.txt` — файли тестових даних для кожної задачі
`ResultAll.txt` — файл протоколу тестування, створиться автоматично після запуску `FnSolutions.py`

# InData1.txt (Задача 1)
Кожний рядок — послідовність цілих чисел, що закінчується 0:
```
3 28 -4 901 666 -25 -25 700 -1 0
```
# InData2.txt (Задача 2)
Кожний рядок — три числа (через пробіл):
```
3 4 5
1 1 1
-1 2 3
```
# InData3.txt (Задача 3)
Кожний рядок — речення для розбиття на слова:
```
Це тестове речення.
```
# InData4.txt (Задача 4)
Кожний рядок — матриця NxN (рядки через ;, елементи через пробіл):
```
1 2 -1;0 -2 3;4 0 -1
```

Для кожної задачі реалізовано різні види тестів: функціональні, деструктивні, тести зручності.

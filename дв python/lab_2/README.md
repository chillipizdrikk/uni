Інструкція

Призначення продукту:
Програма допомагає зручно переглядати, аналізувати та зберігати показники погоди для різних міст на основі даних, отриманих із текстового файлу. Програма не потребує доступу до інтернету під час роботи і забезпечує швидкий пошук потрібної інформації.

Функціональні можливості:
- Перегляд детальних погодних показників для обраного міста.
- Пошук міста з найвищою чи найнижчою температурою.
- Аналіз зміни температури у певному місті протягом дня.
- Визначення переважаючого напрямку вітру серед міст.
- Виявлення міст, де температура нижча за задану.
- Збереження результатів аналізу у текстовий файл.

Інструкція з використання:
1. Для роботи програми потрібен файл з даними погоди (`weather_data.txt`). Усі дані мають бути записані у відповідному форматі(місто TAB дата TAB час TAB температура TAB тиск TAB вітер).
2. Виконайте бажану операцію, слідуючи вказівкам програми.
3. Результати аналізу можна зберегти у текстовий файл для подальшого перегляду.

Джерело даних: дані для тестування отримані з сайтів https://www.gismeteo.ua/, https://ua.sinoptik.ua/,https://meteostat.net/ або інших відкритих джерел з прогнозом погоди.

Увага! Редагуйте програмний код самостійно - заборонено. Для зміни чи додавання нових даних редагуйте лише файл з даними погоди.

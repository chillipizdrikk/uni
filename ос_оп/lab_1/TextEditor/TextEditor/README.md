# Текстовий редактор (Text Editor)

Лабораторна робота 1: Програма для перегляду і редагування текстових файлів з підтримкою різних кодувань.

## Можливості

Підтримка кодувань:
- UTF-8 (Unicode)
- Windows-1251 (ANSI для кирилічних мов)
- ASCII

Функції редактора:
- Відкриття текстових файлів
- Збереження з вибором кодування
- Автоматичне визначення кодування при відкриванні
- Ручний перехід між кодуваннями з перечитуванням файлу
- Стандартні операції: Undo/Redo, Cut/Copy/Paste
- Пошук і заміна тексту
- Перевірка на збереження змін при закритті

## Вимоги до системи

- Windows 7 або новіше
- .NET 6.0 Runtime або .NET Framework 4.7.2+
